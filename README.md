# NunuJal

대학생을 위한 팀 프로젝트 협업 도구
객관적인 기여도 산정
태스크 관리

## Stack

- Frontend: React + TypeScript + Vite
- Backend: FastAPI
- Database: PostgreSQL

## Structure

- `frontend`: React web app
- `backend`: FastAPI server
